```python
# 1번째 데이터 추출 (loc함수 사용)
titanic.loc[0,:]
```

### 난수를 이용하여 train/ test셋을 분할 (70%/30%)
random.sample 함수를 이용한다


```python
import random

random.seed(10) # seed숫자 지정
N = len(titanic) # 자료의 행의 수
per_test = 30 # test data의 비율

# test 데이터의 행의 숫자 지정
test_index = random.sample(range(0,N),N*per_test//100) # // 연산자는 몫을 뜻함

# train 데이터의 행의 숫자 지정
train_index = list(set(range(0,N)) - set(test_index))

# train/ test데이터 생성
titanic_test = titanic.loc[test_index,:]
titanic_train = titanic.loc[train_index,:]
```
